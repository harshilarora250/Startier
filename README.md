# Startier

Welcome and thank you for reading the Startier README. You can use the bot anywhere in the HackClub slack workspace, and in most channels.

## Commands that you can use!

### `/starty-ping`

Ping the bot to make sure it's working fine.

### `/giveideastartier`

Give's you a project idea from a special hand-curated (with a tiny bit of AI) list

### `/startiermeme`

Gives you a startup meme lol you'll laugh (probably) EXTREME DANGER CHOKING HAZARD ⚠️

### `/howmakememe`

This command tells you how to make a meme

## Usage 

1. Open the slack from hackclub
2. Run any of the above commands.
